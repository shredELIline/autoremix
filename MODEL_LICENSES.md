# Model licenses

No neural model weights are distributed in this repository.

The current built-in separation and bridge path is deterministic DSP. It must
not be described as neural or generative AI. A model may be enabled only after
all of the following are committed:

- upstream source-code license;
- weight and dataset license;
- immutable download URL and checksum;
- model card and intended-use limits;
- measured package size, peak RAM, latency, and supported-device matrix;
- a removable provider implementation and deterministic fallback.

See `THIRD_PARTY_MODELS.md` for evaluated candidates.
